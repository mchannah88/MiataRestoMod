# MiataRestoMod
Miscellaneous projects to modernize a 1999 turbo miata

This readme will serve as an overview of the current projects to give the general scope

1. Raspberry pi 4 headunit
   
   a. Raspberry Pi 4B
   
   b. Raspberry Pi Official touchscreen 1
   
   c. Mausberry USB-C 4amp power supply
   
   d. Taramps TS400X4 Amp

Functionality is managed within 2 locations: services and scripts.

These are a mix of python and bash files.

Services should go into the etc/systemd/system/ folder

Scripts should go in /home/$USER$/scripts/
